Odoo - Klaviyo Private API Keys

This module is the base for other Odoo - Klaviyo modules.
Here you can configure private API keys for production and test environments of dependent modules.

Modules that depend on this one:

Configuration:

  1. Login to your Klaviyo.com account and create the private API Key.
    Follow the steps in How to create or clone a private API key: https://help.klaviyo.com/hc/en-us/articles/7423954176283.
  2. After installing the module go to General Settings -> Klaviyo section.
    Enter the production and/or test private API keys created in point 1.
    Enable or disable the test mode switch based on your needs.
